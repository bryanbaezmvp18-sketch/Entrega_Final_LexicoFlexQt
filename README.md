# Analizador Léxico (FLEX + Qt)

Este proyecto corresponde a la implementación de un **Analizador Léxico** utilizando **FLEX** y una **interfaz gráfica desarrollada en Qt**.  
El objetivo es identificar y clasificar tokens a partir de un texto de entrada, cumpliendo con los requisitos académicos establecidos.

 Características del proyecto
- Analizador léxico implementado con **FLEX**
- Aplicación **gráfica** desarrollada con **Qt** (no consola)
- Identificación de tokens y lexemas
- Visualización de resultados en una tabla
- Ejecutable portable para Windows
- 
 Lenguaje de prueba (MiniLang)
El analizador reconoce elementos básicos de un lenguaje de programación, tales como:
- Palabras reservadas (`if`, `else`, `while`, etc.)
- Identificadores
- Números enteros y decimales
- Operadores aritméticos y relacionales
- Delimitadores y separadores
- Cadenas de texto

 Ejecución del programa
1. Descargar el archivo **`Entrega_Final_LexicoFlexQt.zip
2. Descomprimir la carpeta
3. Ejecutar **`LexicoFlexQt.exe
4. Escribir el código de prueba en el área de entrada
5. Presionar el botón **Analizar
